# Alacritty

Terminal emulator. Config: `alacritty/.config/alacritty/alacritty.toml`.

---

## Custom Keybindings

| Key | Action |
|-----|--------|
| `Shift-Enter` | Send `Esc + CR` — acts as newline-without-submit in TUIs like Claude Code |

---

## Config Notes

- **Window:** no decorations, starts maximized
- **Font:** JetBrainsMono Nerd Font, 12pt
- **Colors:** custom Ayu-style palette (background `#0a0e14`, foreground `#b3b1ad`)
