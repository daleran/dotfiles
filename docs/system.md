# System Shortcuts

---

## GNOME Window Management

| Key | Action |
|-----|--------|
| `Alt-F4` | Close window |
| `Alt-F10` | Toggle maximize |
| `Super-h` | Minimize window |
| `Alt-Space` | Window menu |
| `Alt-Tab` / `Shift-Alt-Tab` | Switch windows (forward / reverse) |
| `Super-d` | Show desktop |

---

## GNOME Workspaces & Apps

| Key | Action |
|-----|--------|
| `Ctrl-Alt-Up/Down` | Switch workspace up / down |
| `Ctrl-Shift-Alt-Up/Down` | Move window to workspace |
| `Super-1`–`9` | Switch to app on dock |
| `Super-a` | Show all apps |
| `Super-s` | Quick settings |
| `Print` | Screenshot (UI) |
| `Alt-Print` | Screenshot (window) |

---

## Chrome / Browser

| Key | Action |
|-----|--------|
| `Ctrl-l` | Focus address bar |
| `Ctrl-t` / `Ctrl-w` | New tab / close tab |
| `Ctrl-Tab` / `Ctrl-Shift-Tab` | Next / prev tab |
| `Ctrl-f` / `Ctrl-g` | Find / find next |
